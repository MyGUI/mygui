---------------------------------------------------------------------------------------
версия - 2.2.2
---------------------------------------------------------------------------------------

-- Core --
- преобразована иерархия виджетов - добавлены стили:
   * Child - дочерний виджет, обрезается по границам окна и не перекрывается (по умолчанию у дочерних)
   * Popup - всплывающий виджет, имеет родителя, но не обрезается по нему, а свободно перемещается
   * Overlapped - дочерний виджет, обрезается по границам родителя (если он есть), и может перекрываться (у рутовых по умолчанию)
- поддержка отсоединения и просоединения виджетов, смена скинов и смена стилей на лету
- введена быстрая система RTTI (идентификация типа во время выполнения). Теперь типы виджетов можно безопасно проверять


-- Widgets --
- добавлен виджет DDContainer - базовый класс для всех виджетов имеющих дроп
- добавлен виджет MenuCtrl - базовый класс для всех меню
- добавлен виджет MenuBar
- добавлен виджет ScrollView
- Widget имеет поддержку телтипов
- StaticImage поддерживает анимацию


-- Managers --
- добавлен менеджер ResourceManager - позволяет загружать пользовательские ресурсы, для примера ResourceImageSet
- добавлен менеджер LanguageManager - позволяет автоматически заменять теги в загружаемых лейаутах по загруженным словаря, а так же обеспечивает ручную вставку и замену тегов
- менеджер PointerManager - поддерживает анимированные курсоры
- менеджер FontManager - поддерживает загрузку кастомных шрифтов из текстуры


-- Controllers --
- расширены контроллеры движения - добавился Jump


-- Resources --
- добавлен тип ресурса ResourceImageSet позволяет создавать группы картинок, с поддержкой анимации, загружается через xml


-- SubWidgets --
- добавлен сабвиджет RawRect для прямого доступа к вершинам квада
- добавлен сабвиджет TileRect для поддержки тайлинга квада


-- Others --
- все виджеты контейнеры имеют Any (аналогичен boost::any) для хранения пользовательской информации об айтеме
- добавлено указании версии для xml
- добавлены две темы для скинов


-- Wraps --
- BaseLayout - базовая обертка, для загружаемых лейаутов
- BaseItemBox - обертка, для удобного и быстрого использования ItemBox а
- добавлен контрол PanelView - полностью сделан на обертках

 
-- Tools --
- расширен LayoutEditor, поддержка внешних ресурсов для удобной интеграции с внешними проектами, ряд улучшений интерфейса
- добавлен ImageSetViewer для просмотра ресурсов типа ResourceImageSet - наборы картинок с поддержкой анимации


-- Demos --
- Demo_Colour - контрол управления цветом, сабвиджет RawRect и управление цветом текста в EditBox
- Demo_Console - консоль для управления виджетом
- Demo_Controllers - контроллеры движения и альфы для управления виджетами
- Demo_Font - генерация шрифта truetype с возможностью сохранитьв формате MyGUI для быстрой интеграции шрифта
- Demo_Gui - совместное использование виджетов разных стилей и типов (аналогично Ogre Demo_Gui)
- Demo_ItemBox - ItemBox, обертка BaseItemBox, ToolTip и внешний пользовательский ресурс, загружаемый через xml
- Demo_PanelView - контрол на основе обертки
- Demo_Picking - фокус виджета мышью по произвольному контуру
- Demo_PluginStrangeButton - использование внешних плагинов и внешних виджетов
- Demo_ScrollView - скроллинг виджета
- Demo_Themes - множество тем
- Demo_StaticImage - простейший вьювер ImageSet
